Sublime Text3

Instructions

Download

Below are steps to install Quantified Dev Sublime Text 3 Plugin

1. Go to 'View' then click on 'Show Console'

2. Copy following code snippet into the text bar of the console (visible at bottom of window)

import urllib.request, os, hashlib; pf = 'QuantifiedDev.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener(urllib.request.build_opener(urllib.request.ProxyHandler()));by = urllib.request.urlopen('http://app.quantifieddev.org/' + pf.replace(' ', '%20') ).read();open(os.path.join(ipp, pf), 'wb').write(by)

1. Press Enter

2. Go to 'View' and then click on 'Hide console' to hide console.

The plugin will be installed and your Sublime Text 3 activity will be logged by plugin

To go to the QD Dashboard from Sublime Text 3, there are two ways

  • Go to 'Tools' then click on' QuantifiedDev'. Then click on 'Go to QuantifiedDev Dashboard' OR
  • Press CMD + Shift + P on Mac and Ctrl + Shift + P on LInux/Windows then type 'Go to QuantifiedDev Dashboard'

View your events

Quantified dev has a console where you can view your events. First, you need the stream id and read token from the plugin. You can find this in the plugin view. Go to the view menu, other windows and click on quantified dev. Take the stream id and the read token and copy them. Go to quantifieddev.org and put them into the stream id and read token boxes. Press save. Now your development events will appear in the console.

Configure

The quantified dev plugin makes a guess about where you are, based on your IP address. This will normally be accurate to the country level. You can enter your lat and long manually in the plugin to increase the accuracy.
