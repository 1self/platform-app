Sublime Text

Quickstart

1. Open the console

2. Paste in:

For Sublime Text 2

import urllib2,os,hashlib;pf = 'QuantifiedDev.sublime-package';ipp = sublime.installed_packages_path();os.makedirs( ipp ) if not os.path.exists(ipp) else None;urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler()) );by = urllib2.urlopen( 'http://app.quantifieddev.org/' + pf.replace(' ', '%20')).read();open( os.path.join( ipp, pf), 'wb' ).write(by)

For Sublime Text 3

import urllib.request, os, hashlib; pf = 'QuantifiedDev.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener(urllib.request.build_opener(urllib.request.ProxyHandler()));by = urllib.request.urlopen('http://app.quantifieddev.org/' + pf.replace(' ', '%20') ).read();open(os.path.join(ipp, pf), 'wb').write(by)

3. Register by following the link in the QuantifiedDev tool menu

4. How do i see my stats

- Open Sublime Text Editor

- Go to tools → QuantifiedDev → Go to QuantifiedDev Dashboard or CTRL + SHIFT + P → Go to QuantifiedDev Dashboard

IntelliJ IDEA

Download

Quickstart

1. Download the intellij plugin - it works with all versions

2. Open plugin preferences and choose install from disk

3. Register by following the link in the tool window with the QD logo

4. How do i see my stats

- Open Intellij Idea

- Go to File -> Settings(Ctrl + Alt + S) -> Quantified Dev Settings

- Copy streamid

- Go to http://quantifieddev.azurewebsites.net/app

- Paste the streamid in 'streamid' textbox

- Go back to IntelliJIdea and copy read token

- Go back to website and paste the read token in 'read token' textbox

- Click 'Save'.

Now as you build your projects from Idea, you can see corresponding stats reflected on the websitehttp://app.quantifieddev.org/dashboard
